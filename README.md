# Prova prática de gerência de recursos em Linux

Este repositório contém a atividade prática destinada a explorar conceitos de gerência de processador, memória, arquivos e dispositivos no Linux.  
A proposta é que você navegue pelas etapas, execute os comandos e registre suas observações sobre o comportamento real do sistema operacional.

## Como executar

Baixe ou clone o repositório e, em seguida, execute o script inicial:

```bash
cd unifebe-so-pp2
chmod 755 start.sh
./start.sh
```

## Pré-requisitos

- Sistema operacional Linux  
- Terminal Bash  
- Permissão para executar scripts

## Downloads recomendados

- **Ubuntu Server (LTS):** https://ubuntu.com/download/server  
- **PuTTY (SSH para Windows):** https://www.putty.org

## Observação

O script `start.sh` guiará você pelas etapas da prática.  
Siga as instruções exibidas na tela.
